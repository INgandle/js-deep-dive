# js-deep-dive
모딥다와 인간들

### study rules
- 정해진 분량을 읽고, 해당 레포지토리에 정리합니다.
- 오프라인 모임에서 정리 내용을 공유하여 읽어보고, 준비한 질문을 나누며 토론합니다.
- 커밋 형식
  ```
  :본인이 하고 싶은 깃모지: n장: n장의 제목
  ```
- 파일 이름
  ```
  n장_n장제목_이름.md
  ```
- 폴더 구조
  - 각자 이름으로 된 폴더를 생성하고, 그 안에 장마다 md 파일을 생성하여 정리합니다.
